Belief and practices

I am a staunch believer in the use of Open -source software.

Open source software is software with source code that anyone can inspect, modify, and enhance. This system of open-source software allows for faster detection, assessment and removal of bugs and vulnerabilities while allowing the software to grow at exponential rate due to other programmers who are also using this software for their own projects helping each other out in creating a better environment for software development in the future for each other.

I also practice SWOT analysis for planning for future projects and keeping a realistic deadline. SWOT analysis is a strategic planning technique used to help a person or organization identify the Strengths, Weaknesses, Opportunities, and Threats related to project planning.
